# Privacy Policy — Zoom Chat Auto-Responder
> 🌱 この拡張機能はプログラミングの練習として初めて作ったものです。
> 温かい目で見ていただけると嬉しいです。
**最終更新日：2026年4月16日**

## 収集するデータ

本拡張機能はユーザーデータを一切収集・保存・送信しません。

## 保存される情報

ユーザーが設定した自動返信ルール（トリガーワード・返信文・
しきい値など）は、Chromeの同期ストレージ（chrome.storage.sync）に
**ユーザーの端末内にのみ**保存されます。
開発者はこの情報に一切アクセスできません。

## Zoomチャットメッセージの取り扱い

Zoomのチャットメッセージはトリガーワードの検出のために
端末内でリアルタイム処理されます。
メッセージの内容は保存・記録・外部送信を一切行いません。

## 第三者への情報提供

いかなる第三者にも情報を提供しません。

## お問い合わせ

ご質問はGitHub Issuesまでお寄せください。
